# Discipline — Goal Tracker

A personal habit/discipline tracker, in two forms:

- **Web dashboard** (`index.html`, `style.css`, `app.js`) — the full-featured version, runs entirely in your browser.
- **Excel template** (`Goal Tracker Template.xlsx`) — a simpler version you can email to a friend who wants to track their own goals.

Both default to the same 8 habits: Wake Up at 6:00 AM, Meditation / Stretch, Hit the Gym, Cold Shower, Read 10 Pages, Pray, Deep Work, Limit Social Media.

## Web dashboard

Open `index.html` directly in a browser (Chrome or Edge recommended) — no server or install needed.

- **Daily tracking** — click a cell in the grid to check off a habit for that day. Daily Score, streaks, and every chart/stat recompute instantly.
- **Mental State / Motivation** — pick 1–10 from the dropdowns in the last two rows of the grid each day. These feed the two comparison line charts at the bottom.
- **Dashboard widgets** — Daily/Weekly/Monthly progress bars, current streaks (with 🔥), habit heatmap, and a full completion calendar.
- **Edit Goals** (gear button) — rename, reorder, add, or remove habits at any time. Everything recalculates live. Removing a habit drops its check marks for the current month; renaming preserves history.
- **Export / Import** — Export downloads your full tracker data as JSON (handy for backups or moving between devices). Import loads a previously exported file.
- **Monthly reset** — the first time you open the dashboard in a new calendar month, the previous month is automatically archived (pick it from the dropdown next to the title to view it read-only) and you get a fresh grid.

All data is stored in your browser's `localStorage` — nothing is sent to a server. Clearing browser data for this page will erase your history, so export a backup periodically if you care about long-term records.

### Use it on your phone

This dashboard is hosted live at **https://ncmoore55.github.io/goal-tracker/** via GitHub Pages, so you can use it on your phone without running anything locally.

1. Open that URL in your phone's browser (Safari on iPhone, Chrome on Android).
2. Tap the share icon and choose **Add to Home Screen**.
3. It launches full-screen like a real app — no browser address bar — with the "Discipline" name and icon.

Note: your phone and any other device each keep their own independent `localStorage` data — there's no syncing between them. Use Export/Import if you want to move your data from one device to another.

## Excel template

Open `Goal Tracker Template.xlsx` in Excel (or LibreOffice/Google Sheets — conditional formatting and charts may render slightly differently outside Excel).

Everything lives on a single **Tracker** sheet, top to bottom, sized and zoomed (80%) to fit one screen width — no horizontal scrolling needed.

1. Set **Month** (B2) and **Year** (D2) near the top.
2. Rename the habits in column **A** (rows 8–15) to whatever goals you're tracking.
3. **Daily Tracking** — each day cell is boxed like a checkbox. Click it and choose **✓** from the dropdown (or just type `✓`) to mark a habit done. Leave it blank for an incomplete day. Checked cells turn green with a larger checkmark.
4. Fill in **Mental State** and **Motivation** (rows 17–18) each day, 1–10, from the dropdown.
5. Completion %, Current Streak, and Best Streak (columns B–D) update automatically.
6. Scroll down for **Weekly Tracking** (one row per Sun–Sat week, with the current week highlighted in amber), **Monthly Tracking** (overall stats), then the **Progress Charts** — bar charts for habit completion % and weekly score progression, plus pie charts for habit completion share and this month's completed-vs-incomplete check-ins.

Notes:
- The grid has 31 day columns; for shorter months, just ignore the extra columns past "Days in Month" (shown near the top of the sheet) — formulas already account for the real month length.
- The **Helpers** sheet is hidden — it holds the streak/weekly-breakdown and chart-data calculations. Don't edit it, but you can unhide it (right-click a sheet tab → Unhide) if you're curious how the formulas work.
- Checking a box still takes two steps (click the cell, then pick ✓ from the dropdown) — Excel's native one-click checkboxes require a newer Excel 365 feature that can't be reliably generated from a script. If you have Excel 365, you can select the day-grid range and use **Insert → Checkbox** yourself for true one-click toggling.
- This file has no macros and no external links — safe to email.

## Regenerating the Excel template

The Excel file is generated by `generate_excel_tracker.py`. To customize the default habit list (e.g. before sending a fresh copy to someone else) or regenerate it after changing the script:

```
pip install openpyxl
python generate_excel_tracker.py
```

This overwrites `Goal Tracker Template.xlsx` in this folder. Edit the `HABITS` list near the top of the script to change the default goal names — the recipient can still rename them directly in Excel afterward without rerunning anything.
